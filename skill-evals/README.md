## Criterio

Las evaluaciones pueden ser:

- Documentales: revision manual de disparadores, alcance y conflictos.
- Cualitativas: comparacion razonada de respuestas esperadas.
- Automatizadas: ejecucion con subagentes, scripts o benchmarks cuando este disponible y justificado.

No se deben inventar metricas. Si no se ejecuta una evaluacion automatizada, debe indicarse.

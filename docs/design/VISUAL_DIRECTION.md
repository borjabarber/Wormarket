# Direccion visual

Wormarket debe sentirse como un mercado funcional atravesado por fenomenos interdimensionales.

## Principios

- Marketplace real primero, fantasia como capa expresiva.
- Interfaz clara, escaneable y accesible.
- Detalles visuales inspirados en portales, coordenadas, rareza y comercio dimensional.
- Evitar una estetica generica de landing futurista.

## Tono

- Preciso.
- Ligeramente fantastico.
- Confiable.
- En espanol para el usuario final.

## Imagenes

Los productos deben mostrarse claramente. Cuando no existan imagenes reales, usar assets generados o placeholders semanticos que permitan entender el objeto.

Los avatares demo deben sentirse como perfiles reales del marketplace, pero con rasgos interdimensionales sutiles. Deben ser personajes ficticios, no parecerse a personas reales ni copiar personajes existentes.

## Movimiento

Usar microinteracciones con moderacion y respetar `prefers-reduced-motion`.
